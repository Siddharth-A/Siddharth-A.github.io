SAP-1 ARCHITECTURE

8 bit breadboard computer using 74LSXX series IC's

Project Overview

I stumbled upon the idea of a SAP (simple as possible) breadboard computer while surfing Youtube one day and thought it would be a great idea to replicate the computer myself and improve upon the initial design. The computer is composed of 13 independent modules and can be programmed with basic machine language using the RAM dip switches to do simple math.

Key Learnings

  • Computer architecture
  • Digital circuit design
  • Flip-flops, latches, logic-gates, clocks, EEPROMs, RAMs
  • B.O.M management, project timeline, cost-effective design strategies

Timeline: December 2017 - February 2018

Category: Digital design, Computer architecture, ASIC
